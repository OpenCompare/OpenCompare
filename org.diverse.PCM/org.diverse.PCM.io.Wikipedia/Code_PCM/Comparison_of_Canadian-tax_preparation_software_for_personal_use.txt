A list of personal use [[Tax return (Canada)|Canadian income tax preparation]] software packages and Web applications.  They are listed from cheapest to most expensive.  Many allow users to submit returns using the [[Canada Revenue Agency]]'s (CRA) [[NETFILE]] service while others allow users to print forms that may or may not be CRA approved or have a 2D [[barcode]].  2D bar coded returns are scanned by the CRA while others are keyed.
