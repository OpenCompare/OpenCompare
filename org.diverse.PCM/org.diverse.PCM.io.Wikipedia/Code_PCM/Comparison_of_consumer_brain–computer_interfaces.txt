==Open-source projects==

[http://github.com/daeken/Emokit Emokit] is an open-source Python library for reading out sensor data from the EPOC ([[Emotiv Systems]]) by [[Cody Brocious]]. It was built by reverse-engineering the encrypted protocol.<ref name=Emokit>[http://github.com/daeken/Emokit/blob/master/Announcement.md Python library for the Emotiv EPOC headset] on Github</ref> Emokit has been deprecated in favour of [http://github.com/qdot/emokit emokit].<ref name=emokit>[http://github.com/daeken/Emokit/blob/master/README.md daeken/Emokit has been deprecated in favour of qdot/emokit] on GitHub</ref>

In 2011 ''[[Make (magazine)|Make]]'' magazine published an article on [http://blog.makezine.com/archive/2011/05/behind-the-brain-blinker.html hacking NeuroSky headsets]. In 2012 [[Hack a Day]]  published an article on  [http://hackaday.com/2012/12/20/modifying-an-eeg-headset-for-lucid-dreaming/ modifying NeuroSky headsets for sleep and dream research].

Open-source Matlab toolboxes such as [[EEGLAB]], Fieldtrip, and the [[Neurophysiological Biomarker Toolbox|Neurophysiological Biomarker Toolbox (NBT)]] can be used to process data from the [[electroencephalography|electroencephalography.]] The Fieldtrip toolbox also offers a real-time plugin.<ref>{{cite web|url=http://fieldtrip.fcdonders.nl/development/realtime}}</ref>

[[OpenVibe]] is a LGPL software platform (C++) to design, test and use BCI.<ref name=openvibe>[http://openvibe.inria.fr home of the OpenViBE software]</ref> The software comes with an acquisition server that is currently compatible with many EEG device including Neurosky Mindset, Emotiv EPOC (Research Edition or above) and OpenEEG. The software is developed at [[INRIA]].

Several open-source computer programs are also available from [[EPFL]]'s CNBI project.<ref>[http://cnbi.epfl.ch/software/index.html EPFL CNBI project]</ref><ref>[http://www.kurzweilai.net/nissan-teams-up-with-epfl-for-bmi-assisted-driving EPFL CNBI project]</ref>

The openEEG project<ref>{{cite web|title=OpenEEG project|url=http://openeeg.sourceforge.net/doc/}}</ref> has developed several open hardware EEG devices while the openBCI project<ref>{{cite web|title=OpenBCI|url=http://www.openbci.com/}}</ref> plans to release an open hardware device in 2014.

==Technology==
All of the devices listed use [[electroencephalography]] except the Neural Impulse Actuator which only uses [[electromyography]].
Some use both electroencephalography and electromyography, such as the BrainBand MindWave, EPOC, and MindSet.
