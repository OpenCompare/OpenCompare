'''Network Mapping and Drawing Software'''

A number of software tools exist to design [[computer network]] diagrams. Broadly, there are two types of software tools:-

* Network Mapping Software

* Drawing Tools
Network maps support IT systems managers to understand what hardware and software services are on the network and how they are interconnected so that they may better manage the systemâ€™s uptime, performance, security risk and support to changes.
