==NOTE==
[http://spring.io/tools/sts Spring Tools Suite] and [http://www.eclipse.org/ Eclipse] have a variety of built-in and free plugins matching or exceeding the quality of the paid versions. They come with a content assist tool that completes tags and can search for classes in any java classpath. They also include validation, bean creation, and commit tools.

A plugin was also developed for [http://notepad-plus-plus.org/ Notepad++] named [[:sourceforge:apps/mediawiki/notepad-plus/?title=Plugin Central|XML Tools]] that contains many features including manual/automatic validation, xpath evaluation, auto-completion, pretty print, and text conversion in addition to being able to work on multiple files at once. Other tools are available to run xhtml.
