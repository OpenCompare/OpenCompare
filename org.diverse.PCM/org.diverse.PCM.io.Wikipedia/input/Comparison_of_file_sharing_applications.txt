
[[File sharing]] is a method of distributing [[electronically stored information]] such as [[computer program]]s and [[digital media]]. Below is a list of file sharing applications, most of them make use of [[peer-to-peer file sharing]] technologies.

This comparison contains next to [[file sharing]] applications also [[download manager]]s that can be used as file sharing applications (using the torrent-protocol). For pure download managers see the [[Comparison of download managers]].
